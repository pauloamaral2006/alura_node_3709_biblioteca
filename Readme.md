# Node.js

## ℹ️ Sobre

<p>Projeto utilizado no curso para aprendizado das técnicas e melhores práticas.</p>

## 📘Ementa

### JavaScript com Node.js: criando sua primeira biblioteca

- Praticar lógica de programação com JavaScript resolvendo um problema real
- Criar seus próprios comandos para serem executados no terminal
- Entender o que são promessas e como trabalhar com código assíncrono em JavaScript
- Aprender como utilizar bibliotecas nativas do Node.js para interagir com o sistema de arquivos do computador
- Aprender a importar e utilizar métodos de repositórios de código aberto
